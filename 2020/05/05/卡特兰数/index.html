卡特兰数

发表于 2020-05-05 | 分类于 卡特兰数

卡特兰数是常出现在计数问题中的数列。
h(n) = h(0) * h(n-1)+h(1) * h(n-2) + … + h(n-1) * h(0) (n>=2)、(h(0) = 1)
可化简为
h(n) = h(n-1) * (4n-2)(n+1)
通项公式 h(n) = C(2n,n)-C(2n,n+1)
可以化为 h(n)=C(2n,n)/(n+1)

前几项为：
1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796, …
在某些算法问题中得到很好的应用。

卡特兰数增长速度较快，int型无法满足，对于Java可以使用 BigInteger来计算大整数
使用化简后的h(n) = h(n-1) * (4n-2)(n+1)

进出栈序列

描述：n个元素进栈序列为：1、2、3、4、…、n,有多少种出栈序列。
思路：令n个元素进出栈问题的解为f(n),假设最后一个出栈的元素为K,则在K之前，进栈并出栈的元素有K-1个，这是一个 K-1个元素进出栈的子问题。在K之后进栈并出栈的元素有n-K个，这是一个n-K个元素进出栈的子问题。即在第K个元素是最后一个出栈的元素时，问题的解法有f(K-1) * f(n-K)个
因为K的取值可以为1,2,3,…,n中的任何一个。即f(n) = f(0) * f(n-1) + f(1) * f(n-2) + … + f(n-1) * f(0)。是一个显然的卡特兰数问题。

括号序列

描述：n对括号，求有多少种匹配的括号序列

左括号视为进栈，右括号视为出栈，和进出栈问题相同。

满二叉树数量

描述：n+1个节点能构成多少个不同的满二叉树。
思路：深度优先搜索这棵满二叉树，向左扩展视为进栈，向右扩展视为出栈

不同路径

一个机器人位于一个 m x n 网格的左上角机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角问总共有多少条不同的路径？

定义dp[i][j] 为机器人从左上角走到[i][j]时共有多少条不同路径，本题答案右下角即为dp[m-1][n-1]。
状态转移方程：
走到[i][j]有两种状态 1、先走到[i][j-1]再往右一步，或2、先走到[i-1][j]，再往下一步。即dp[i][j] = dp[i][j-1]+dp[i-1][j]
初始值：
注意：dp[0][0]并不为0：只有一个格子的时候默认有一条路径而不是0条。
以下两种情况为机器人只一直往右或一直往下
dp[0][0..1..2..n] = 1;
dp[0..1..2..n][0] = 1;

最小路径和

给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。
说明：每次只能向下或者向右移动一步。

定义dp[i][j]为走到这一步时的最小路径和。
状态转移方程：dp[i][j] = Math.min(dp[i][j-1],dp[i-1][j]) + arr[i][j]
初始值：
dp[0][j] = dp[0][j-1]+arr[0][j];
dp[i][0] = dp[i-1][0]+arr[i][0];

编辑距离

给你两个单词 word1 和 word2，请你计算出将 word1 转换成 word2 所使用的最少操作数 。
你可以对一个单词进行如下三种操作：
插入一个字符
删除一个字符
替换一个字符
定义状态dp[i][j] 为 word1的前i个字母转换成word2的前j个字母所需要的最少操作。
则状态转移方程：
当word1[i] == word2[j]时 dp[i][j] = dp[i-1][j-1]
当word1[i] != word2[j]时 有三种可能性：
1、在word1[i]之后插入word2[j],问题变为求word1[i]和word2[j-1]之间的最少操作+1
2、将word1[i]删除，问题变为求word1[i-1]和word2[j]之间的最少操作+1
3、将word1[i]替换为word2[j]，问题变为求word1[i-1]和word2[j-1]之间的最少操作+1(替换操作)
即dp[i][j] = Math.min(dp[i][j-1],dp[i-1][j],dp[i-1][j-1])+1;

初始值：dp[0][0] = 0;即两个都是空单词，无需操作
dp[0][1..2..3..m] = m;
dp[1..2..3..n][0] = n;
即两个单词中有一个为空，若word1为空，对其进行word2长度的插入即可，若word2为空，对其进行word1长度的删除。
此题需要注意单词位数和数组下标之间的对应

强盗抢劫

不能抢相邻的住户，求抢劫金额的最大值
定义状态转移方程dp[i]为抢到第i个住户的最大金额
若抢了 i-1 用户，则不能抢i用户
若只抢到i-2用户，则可以抢i用户
那么
dp[i] = Math.max(dp[i-2]+nums[i],dp[i-1]);
初始值：
dp[0] = nums[0];
dp[1] = Math.max(dp[0],nums[1]);
注意要将数组为空和nums数组只有一个值的情况当做特例先解决，否则dp数组会溢出。

信件错排

有N个信和信封，它们被打乱，求错误装信方式的数量,要求每封信都不在其原来位置。
状态转移方程：
N个元素的错排可分两步完成：
1、先把第1个元素排在第2个到第N个位置，有n-1种排法
2、当第1个元素被排在第k个位置时，再来排第k个元素，此时分为两种情况
2.1、第k个元素被排在第1个位置，那么剩下的问题是n-2个元素的错排问题，其解为 f(n-2)
2.2、第k个元素没有被排在第1个位置,待错排的元素有n-1个,可被错排的位置有n-1个,即剩下的问题是n-1个元素的错排问题，其解为f(n-1)
可得f(n) = (n-1) * (f(n-1) + f(n-2))
初始值：f(1) = 0, f(2) = 1, f(3) = 2

母牛生产问题

假设农场中成熟的母牛每年都会生1头小母牛，并且永远不会死。第一年有1只小母牛，从第二年开始，母牛开始生小母牛。每只小母牛3年之后成熟又可以生小母牛。给定整数 N，求 N 年后牛的数量。

这题和很著名的斐波那契兔子问题类似
状态转移方程：每一年的牛数量 = 去年的牛数量 + 新增的牛数量
每年新增的牛数量 = 当年成熟母牛数量 = 三年前的母牛数量
即 f(n) = f(n-1) + f(n-3)
初始值：f(1) = 1,f(2) = 2,f(3) = 3;

leetcode 303 区域和检索 - 数组不可变

给定一个整数数组nums，求出数组从索引i到j(i ≤ j)范围内元素的总和，包含i,j两点。
思路：新建一个数组sums，sums[i]为nums数组从nums[0]累加至nums[i-1]的值。
【开始想的是sums[i]为nums数组从nums[0]累加至nums[i]的值，但这会导致写出sums[i] = sums[i-1]+nums[i]这样一个下标不相等(i-1 和 i)的式子出来,极易发生越界等错误】
计算从i到j的值时 即可用sums[j] - sums[i-1]快速求解得出

leetcode 746. 使用最小花费爬楼梯
数组的每个索引做为一个阶梯，第 i个阶梯对应着一个非负数的体力花费值 costi。
每当你爬上一个阶梯你都要花费对应的体力花费值，然后你可以选择继续爬一个阶梯或者爬两个阶梯。
您需要找到达到楼层顶部的最低花费。在开始时，你可以选择从索引为 0 或 1 的元素作为初始阶梯。

状态转移方程：f(i) = cost[i] + Math.min(f(i-1),f(i-2))

leetcode 1025. 除数博弈

爱丽丝和鲍勃一起玩游戏，他们轮流行动。爱丽丝先手开局。
最初，黑板上有一个数字 N 。在每个玩家的回合，玩家需要执行以下操作：
选出任一 x，满足 0 < x < N 且 N % x == 0 。
用 N - x 替换黑板上的数字 N 。
如果玩家无法执行这些操作，就会输掉游戏。
只有在爱丽丝在游戏中取得胜利时才返回 True，否则返回 false。假设两个玩家都以最佳状态参与游戏。

常规解法：
play[i]记录了数为i时的结果，true为胜利，false为失败。

数学解法:
最终结果是取到2的必然胜利，取到1的失败。
若当前数为奇数，其约数必然为奇数，则取走后变为偶数。
若当前数为偶数，只需一直取1，让取完后的数字变为奇数，即可让对方不断地给自己提供偶数。
最终自己能取到最小偶数2，获得胜利。
即 偶数胜利，奇数失败
